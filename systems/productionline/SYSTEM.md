# ProductionLine

## Overview

Design-to-app-store factory for mobile products. It creates designs, generates platform code, tests on devices, and prepares release assets for both iOS and Android.

## Core Flow

1. Convert the product brief into a design system and UI assets.
2. Sync tokens into SwiftUI and Android implementations.
3. Test the product on real devices and validate accessibility.
4. Package the release and generate the supporting docs.
