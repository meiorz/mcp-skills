# OrbitQA

## Overview

Multi-agent software quality team for pull requests. It runs architecture, implementation, testing, security, documentation, memory, and release checks in parallel before merge.

## Core Flow

1. Review the PR against architecture and product constraints.
2. Apply fixes through a patch-based editing workflow.
3. Execute browser, visual, and security checks in parallel.
4. Update docs and release artifacts, then record the outcome in persistent memory.
