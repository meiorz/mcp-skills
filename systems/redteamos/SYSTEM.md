# RedTeamOS

## Overview

Autonomous security audit platform for repos and domains. It combines OSINT, static review, threat modeling, compliance checks, and reporting into a single analysis pipeline.

## Core Flow

1. Run recon and exposure checks in parallel.
2. Map code ownership and attack surface to likely blast radius.
3. Correlate code, dependency, API, and infrastructure findings into one threat model.
4. Produce a technical report and a concise executive summary.
